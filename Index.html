About Niva Mesh

Niva Mesh provides resilient civilian-to-rescuer connectivity when infrastructure fails, using BLE mesh networks, hop-forwarding, clustering, and localization algorithms. Our framework ensures critical communication capabilities during disasters when traditional networks are compromised.

Ultra-Low Power

Optimized BLE communication with energy cost of only 2.9 mJ per message.

Decentralized Network

No infrastructure dependency - devices form autonomous mesh networks.

Extended Range

Multi-hop forwarding extends communication range beyond single device limits.

Resilient Design

Self-healing network topology adapts to device failures and mobility.

System Architecture

Four core components work together to create a resilient mesh communication framework.

BLE Mesh Formation

Devices automatically discover and connect to nearby nodes using Bluetooth Low Energy protocols

Auto-discovery Dynamic topology Connection management

Hop-based Forwarding

Messages are relayed through multiple nodes to extend communication range beyond single-hop limits

Multi-hop routing Path optimization Loop prevention

Clustering & Aggregation

Network nodes organize into clusters for efficient resource management and data aggregation

Cluster formation Data aggregation Load balancing

Rescuer Triangulation

Emergency responders can locate civilians using network-based positioning and signal triangulation

Position estimation Signal strength mapping Location services
